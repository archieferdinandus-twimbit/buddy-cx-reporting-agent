# Insight Synthesis Frameworks

## Framework 1: Evidence-Based Comparison
1. State the finding ("Company X leads in Digital")
2. Cite the score as evidence (digital_score = 4.7)
3. Explain with qualitative context ("Their mobile app offers...")
4. Compare to peers ("This exceeds the industry average of 3.2")

## Framework 2: Theme Extraction
1. Retrieve 10-15 qualitative chunks on the topic
2. Group into 3-5 recurring themes
3. For each theme: cite 2-3 company examples
4. Rank themes by frequency and impact

## Framework 3: Gap Analysis
1. Identify the highest and lowest performers on a criterion
2. Retrieve qualitative evidence for both
3. Articulate what the leader does that the laggard does not
4. Quantify the gap (score difference, band difference)

## Tone
- Analytical, not promotional
- Use "the data shows" not "I think"
- Be specific: name companies, cite scores, quote practices
