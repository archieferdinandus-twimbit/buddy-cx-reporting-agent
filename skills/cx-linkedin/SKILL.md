# CX LinkedIn Post Generator

## When to use
- "Create a LinkedIn post about this report"
- "Draft a social post highlighting the top findings"
- "Write an announcement about the Banking CX results"

## Process
1. Identify the most compelling 2-3 findings from the report or data
2. Draft using the format guidelines in [references/post-formats.md](references/post-formats.md)
3. Include specific data points — LinkedIn audiences respond to concrete numbers
4. End with a call-to-action and relevant hashtags

## Key Rules
- 200-280 words optimal length
- No markdown bold — use plain text for LinkedIn
- Bold insight-led hook (no generic openers like "Excited to share...")
- Twimbit brand voice: authoritative, data-driven, practitioner-focused
