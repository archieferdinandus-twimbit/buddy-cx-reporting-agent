# Standard Report Section Order

## Section Template
Each section follows this structure:
1. **Opening statement**: One sentence summarizing the key finding
2. **Evidence block**: 2-3 specific data points (scores, rankings, comparisons)
3. **Qualitative context**: Best practices or observations that explain the scores
4. **Implication**: What this means for the industry/company/country

## Tone Guide
- Authoritative but accessible — like a knowledgeable practitioner
- Use active voice: "Company X leads in Digital" not "Digital is led by Company X"
- Avoid superlatives without evidence
- Use hedging language when data is limited: "the data suggests" vs. "it is clear"
- Never use marketing language or promotional tone

## Formatting
- Use headings (H2, H3) to organize sections
- Use bullet points for lists of findings
- Use tables for comparative data (rankings, scores side-by-side)
- Bold key findings and company names on first mention
- Include the mastery band label in parentheses after scores: "4.3 (Advanced)"
